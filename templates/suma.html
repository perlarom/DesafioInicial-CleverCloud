Suma de Dígitos

Código en Python

                
        def digitsSum(inputInt: int) -> int:
            suma = 0
            for num in str(abs(inputInt)):
                suma += int(num)
            return suma

Ejemplos

  • digitsSum(12345) → 15
  • digitsSum(999) → 27
  • digitsSum(9184501) → 28
